# Punto-de-Venta-con-Python
Este proyecto es una Simulación de un Sistema de Punto de Venta (POS) desarrollado en Python con una interfaz de línea de comandos (CLI). Su objetivo principal fue demostrar la aplicación práctica de los fundamentos de la lógica de programación y la gestión de datos estructurados
La aplicación replica el flujo de trabajo de un supermercado, destacando las siguientes funcionalidades:

Gestión de Transacciones: Simula el ciclo de vida de una venta, desde el escaneo de productos hasta la finalización del pago.

Lógica de Pesaje (Balanza): Implementa algoritmos para calcular el precio final de productos vendidos por peso, un componente crucial en sistemas de inventario real.

Manejo de I/O: Utiliza la CLI para la entrada (código de producto/peso) y la salida (ticket de compra), reforzando el entendimiento de las interacciones básicas del programa.

Cálculo Financiero: Realiza la sumatoria de subtotales, totales y determina el cambio exacto (vuelto) a entregar al cliente.

Valor Técnico: Este proyecto demuestra la capacidad para modelar problemas de negocio del mundo real (como la venta y la gestión de inventario) utilizando únicamente las herramientas básicas de Python.
